#### Run On Repl.It
[![Run on Repl.It](https://repl.it/badge/github/Adiixyz/AdyyBOTZ)](https://repl.it/github/Adiixyz/AdyyBOTZ)
```cmd
Open Console
------------
> npm i
> npm i qrcode
> install-pkg webp
> install-pkg ffmpeg
-------------
Click Run
```

## Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install libwebp
> pkg install ffmpeg
> git clone https://github.com/Adiixyz/AdyyBOTZ.git
> cd AdyyBOTZ
> npm i
> npm i qrcode
> node run
```
